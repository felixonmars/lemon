Concern

Test Concerns are used to organize unit tests in groups, so as to address specific quality assurance objectives.

Attributes

testcase[R]

The test case to which this concern belongs.

description[R]

The description of this concern. Make this as detailed as you wish.

testunits[R]

Unit tests that belong to this concern.

setup[R]

Setup procedure for concern.
